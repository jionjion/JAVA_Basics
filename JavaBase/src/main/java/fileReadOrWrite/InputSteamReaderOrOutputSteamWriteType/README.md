## 简介

`InputStreamReader`类和`OutputStreamWriter`继承了其抽象父类`Reader`和`Writer`,包装或解析了byte流和char流之间的转换.
## 类文件

* `InputSteamReaderOrOutputSteamWriteModel`               简单实现了一个文件的读取拷贝
* RA                                                                  将要被拷贝的文件
* RB                                                                  拷贝生成的文件
