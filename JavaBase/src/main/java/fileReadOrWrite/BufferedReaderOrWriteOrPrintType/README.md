## 简介

该类对`FileReader`和`FileWriter`类进行了包装,为其提供了缓冲,提高了文件的读写效率


## 类文件

* *BufferedReaderOrWriteOrPrintType*                   一个简单的文件copy实现
* BRA                                                                被拷贝的文件A
* BRB                                                                拷贝生成的文件B
* BRC                                                                拷贝生成的文件C
